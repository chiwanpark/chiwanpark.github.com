Description

빅 데이터 분석은 실체가 쉽게 보이지 않고, 요구사항이 자주 변하는 형태가 주를 이룹니다. 따라서 잦은 설계의 변경과 많은 PoC가 수반되어야 합니다. 또한, 하나의 결과를 얻기 위해 다수의 분석 작업이 Workflow를 이루어 수행되는 특징이 있습니다.

Apache Hadoop은 대용량의 데이터를 저비용으로 처리할 수 있다는 장점이 있지만, Hadoop Ecosystem에 익숙하지 않으면 다루기 어렵고 MapReduce 구조로 인해 프로그래밍에 익숙하지 않으면 다루기 힘들다는 단점이 있습니다. 또한, Hadoop Ecosystem에 적용되는 많은 sub-project들은 제각기 작동해 통합적으로 관리하기가 어려우며 서로 다른 sub-project 사이의 Workflow 관리도 힘듭니다.

Flamingo Project는 이러한 문제를 해결하기 위해, Hadoop Ecosystem 위에서 빅 데이터 분석, 개발, 운영 환경을 제공하는 오픈 소스 프로젝트입니다. GUI 기반으로 Workflow를 작성할 수 있으며, 웹 기반으로 인터넷이 되면 빅 데이터 분석을 수행할 수 있게 구성되어있습니다. 현재 삼성전자, 네오위즈 등에서 Flamingo Project를 기반으로 자사 빅 데이터 분석 플랫폼을 구축하고 있습니다.

Roles

  • Flamingo Hadoop Manager에서 Apache Hive를 다루는 기능을 담당하고 있습니다.
    • Apache Thrift를 통해 원격으로 Hive Cluster를 제어하는 기능을 지원합니다.
      • Web-based Hive Ad-hoc Query Editor 기능 제공
      • Remote Hive Query Execution 기능 제공
      • Hive Query Execution History Management 기능 제공
  • Flamingo Hadoop Manager의 사용자 인증, 사용자 별 권한 관리 모듈을 담당할 예정입니다.
    • 웹에서 HDFS의 사용자 구조를 그대로 구현하여, 사용자 별 접근 가능한 파일을 제어할 수 있도록 합니다.
