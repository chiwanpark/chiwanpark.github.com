Description

오픈 마켓 시장의 증가와 소셜 기반 서비스가 폭발적으로 늘어나고 있지만, Seller의 측면에서는 판매 부스팅, 마케팅 효과 확인을 위한 데이터 분석 서비스가 부족한 것이 현실입니다. HoneyBee는 HTTP Log 데이터와 서비스의 데이터베이스 자료를 기반으로 데이터 분석을 수행하고 그 결과를 시각화하여 마케팅에 활용하도록 도와주는 솔루션입니다.

Roles

  • 계산된 데이터를 Chart로 그려주는 Javascript 기반 Visualization 모듈을 담당했습니다.
  • 지표 계산을 위한 Hive Query 설계를 담당했습니다.
