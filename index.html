박치완 (Chiwan Park)

Bio

스물세 살, 남자입니다. 영화와 드라마 보는 것을 좋아하고 커피를 달고 삽니다. 집은 용인이지만, 학교가 너무 멀어 맘에 맞는 친구들과 낙성대에서 자취하고 있습니다. 달달한 음식과 치즈를 좋아하는 (그리고 매운 음식을 못 먹는) 전형적인 어린이 입맛입니다.

최근에 살이 너무 쪄, 운동을 통해 다이어트를 하고 있는데, 아침에 헬스장 가는 게 죽을 맛이라 너무 힘이 듭니다.

연세대학교 지구시스템과학과 3학년에 재학 중이며, 컴퓨터에 관심이 많아 컴퓨터과학과를 복수 전공하고 있고, 교직과정 이수 중입니다.

친구들은 2014년 2월에 졸업하는데, 최소 5학년, 수틀리면 6학년까지 다녀야 하는 상황에서 졸업이 언제 오려나 걱정이 이만저만이 아닙니다.

Interests

서버사이드 프로그래밍을 주로 해왔고, 데이터 분석에 관심이 많습니다. 최근 데이터 시각화에 관심을 가져 멀티미디어를 통해 시각화하는 방법에 대해 고민하고 있습니다.

오픈 소스 프로젝트에도 관심이 많아 개발 결과물을 github, bitbucket 등을 통해 공개하려고 노력합니다. 또한, 오픈 소스 세미나와 컨퍼런스도 참가하고 있습니다.

많은 오픈 소스 프로젝트들의 매뉴얼, 과학 전공자들의 글을 보고 느끼는 바가 있어, 좋은 매뉴얼 작성이나 과학적 글쓰기에도 관심이 많은 편입니다. 글을 짜임새 있고 바르게 쓰는 사람을 대단히 부러워합니다.

Experiences

현재 ‘Anbado Video’라는 팀에서 서버사이드 프로그래머로 활동하고 있으며, KBS K-Player 업데이트 프로젝트에 참여하여 실시간 시청 정보 기반 콘텐츠 추천 시스템을 만들고 있습니다.

2013년 6월까지 미래창조과학부 소프트웨어 마에스트로 3기 연수생이었습니다.

스마트TV용 애플리케이션(Smilepet)을 제작했던 경험이 있으며, 연세대학교 학술정보원 홈페이지, 연세대학교 언더우드 기념도서관 홈페이지 개선 작업 등 기타 말할 수 없는 외주 개발 경험이 조금 있습니다.

고등학교 재학 중에 한국정보올림피아드 장려상 (2009년 7월), 한국정보과학경시대회 대상 (2008년 6월) 수상 경력이 있습니다.

Skills

OS X, Linux 같은 유닉스 계열 운영체제를 다루는 데 어려움이 없으며, Windows 플랫폼도 사용하는 데 문제없습니다.

프로그래밍 언어로는 Java, Python을 즐겨 사용합니다. 하지만, C, C++, Javascript, PHP, HTML, CSS, LESS, Matlab도 사용할 수 있습니다.

웹 개발 경험이 다수 있으며, 주로 Flask, Django, Spring 중 하나를 사용하여 프로젝트를 진행하는 편입니다.

표준적인 SQL 코딩이 가능하며, Hibernate, SQLAlchemy 같은 ORM 프레임워크도 사용하는데 무리가 없습니다.

주 관심사인 데이터 분석 작업을 수행하기 위해 Apache Hadoop 플랫폼을 자주 사용하는 편입니다. Apache Hive, Apache Pig를 다룰 수 있으며, Apache Lucene 같은 색인 라이브러리도 자주 사용합니다. 자주 사용하는 NoSQL 스토리지는 mongoDB입니다.

버전 관리 시스템에 대해 이해를 하고 있으며, DVCS를 사용하는 데 어려움이 없습니다. 주로 git, mercurial을 사용하며, 필요한 경우 subversion도 사용합니다.
