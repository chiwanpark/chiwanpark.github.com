Me

My name is Chiwan Park. I'm a undergraduate student majoring in Earth System Sciences and Computer Science in Yonsei University. I'm a developer interested in scalable machine learning, distributed data processing.

I prefer to use Python, Java or Scala for my projects. Sometimes I use C/C++, Objective-C, Javascript, and PHP if need be.

I'm contributing Apache Flink and trying to contribute other projects. I'm supported by Global Open Frontier Program (Korean) which is program for open-source contributors by NIPA. For participating this program, I'm currently located in Nuritkum Square (Korean), Seoul, Korea. You can see my open-source activities in Github or Open Hub Profile.

There are some articles written by me in this website (Korean). Because I'm familiar with text-based language for writing such as CommonMark (the next generation of Markdown), LaTeX, and reStructuredText, Most of my articles are written in that languages.
