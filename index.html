Me

My name is Chiwan Park. I'm undergraduate student majoring in Earth System Sciences and Computer Science in Yonsei University. I'm developer interested in database, distributed computing, and data processing. I'm a member of Anbado Video (Korean) as a server programmer.

I'm contributing Flamingo Project (Korean) and trying to contribute other projects. You can see my open-source activities in github pages. There are some articles written by me in this website.

You can contact me via email, facebook message.
